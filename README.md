# Lovalis Framework

A modular, normative specification and companion documentation set.

## Structure
- CORE_SPEC: normative LSS modules
- COMPENDIUM: narrative books / compendium (multilingual)
- EXPERT_VIEW: policy/science-oriented explanation and figures
- PUBLIC_VIEW: human-friendly explanation
- APP: companion prototypes (local-first)
- META: build notes, glossaries, master index

## Principles
- Local-first, transparency, non-coercion
- No gamification, no behavioral enforcement
- Derived application rules are subordinate to CORE_SPEC
